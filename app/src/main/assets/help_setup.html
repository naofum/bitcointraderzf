Setup

To be able to use this app, you have to connect it to your Mt.Gox account. There are two possiblities available which you can find in the "Security Center" on MtGox.com. In both cases you need to set the "Get Info" and "Trade" permission.

Application Access: this generates an qr-code which can be scanned inside this app.

Advanced API Key Creation: if you don't want to generate an qr-code, you can create a key-pair and enter both keys in the textfields.

Not sure, if this is the right app for you? Go ahead and start a demo by clicking the button!
