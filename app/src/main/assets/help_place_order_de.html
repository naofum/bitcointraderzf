Order erstellen

In dieser Ansicht können Sie eine Order erstellen. Hierbei können Sie zwischen "Ask" (Verkaufen von Bitcoins) und "Bid" (Kaufen von Bitcoins) wählen.

Die Checkbox "Marktkurs-Order" erstellt eine Order, welche zum aktuellen Marktkurs ausgeführt wird.

Alternativ haben Sie die Möglichkeit, einen Preis und eine Menge für die Order zu wählen. Um den aktuellen Kurs zu nutzen, klicken Sie auf das Symbol neben dem Preiseingabefeld. Für eine automatische Übernahme aller Ihrer Bitcoins, klicken Sie auf das Symbol neben dem Mengeneingabefeld.
