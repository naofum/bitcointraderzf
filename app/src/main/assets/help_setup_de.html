Setup

Um diese App nutzen zu können, müssen Sie eine Verbindung zu Ihrem Mt.Gox Account herstellen. Hierzu gibt es zwei Möglichkeiten, welche beide unterhalb des "Security Centers" auf der Mt.Gox Seite zu finden sind. Es werden jeweils die Rechte "Get Info" sowie "Trade" benötigt.

Application Access: Hier können Sie einen QR-Code erstellen lassen, welchen Sie dann innerhalb dieser App einscannen.

Advanced API Key Creation: Alternativ können Sie in Ihrem Konto ein Schlüsselpaar erzeugen lassen und in die beiden Textfelder eintragen.

Sie wollen die App erst testen? Dann klicken Sie einfach auf den Demo button und nutzen Sie einen Test-Account!
