Trailing stop loss

This page allows you to configure a trailing stop loss which notifies you, if the current price drops below a configured limit.

If the current price rises above the set limit plus the set percentage, the limit is adjusted.

Example:
  • Current price is 105$ and you set a trailing stop loss threshold of 100$ with 5%
  • If the price drops below 100$, you get a notification.
  • If the price rises above 110.25$, the threshold is adjusted to 110.25$.

Further information about trailing stop loss can be found on Wikipedia: Stop orders

Note: you can enable auto-selling in preferences. If this is enabled, Bitcoins are automatically sold if the threshold is reached.
